# HighNHeavy Driver App (Mobile)

This is the mobile application for drivers, built with Expo and React Native.

## Features
- **Available Loads**: View and accept assigned loads.
- **My Trips**: History of completed and upcoming trips.
- **Active Trip**: Real-time monitoring of the current load.
- **Messages**: Contact shippers and carriers.
- **Profile**: Manage account settings and notifications.

## Getting Started

1.  Navigate to the `driver-app` directory:
    ```bash
    cd driver-app
    ```
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Start the development server:
    ```bash
    npx expo start
    ```
4.  Open the app:
    -   Scan the QR code with the **Expo Go** app (Android) or **Camera** app (iOS).
    -   Press `a` for Android emulator or `i` for iOS simulator.

## Configuration
Update `lib/api.ts` with your machine's IP address if testing on a real device to connect to the local backend.
